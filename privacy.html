Privacy Policy

Last updated: August 2025

1. Information We Collect

MultiZeno does not intentionally collect or store personal information. The bot may temporarily process the following data to provide functionality:

  • Server IDs for configuration storage
  • User IDs for command functionality
  • Message content when commands are executed

2. How We Use Information

Any data processed by MultiZeno is used solely for providing the bot’s functionality, including:

  • Responding to commands
  • Storing server-specific settings
  • Providing moderation features

3. Data Storage

MultiZeno does not permanently store personal data. Server configurations and settings are stored only as long as necessary to provide the service. No message content is stored beyond the immediate processing required for command execution.

4. Data Sharing

We do not share data with third parties. All information processed by MultiZeno remains confidential and is used exclusively for the bot’s functionality.

5. Your Rights

You can:

  • Request information about what data is stored for your server
  • Request deletion of your server’s configuration data
  • Opt out of data processing by removing MultiZeno from your server

6. Security

We implement appropriate technical measures to protect any data processed by MultiZeno. However, no method of transmission over the Internet or electronic storage is 100% secure.

7. Children’s Privacy

MultiZeno is not directed at individuals under the age of 13. We do not knowingly collect personal information from children under 13.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page.

9. Contact Us
